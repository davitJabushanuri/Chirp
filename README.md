## Development workflow

We use [yarn](https://yarnpkg.com/) as our package manager, so make sure to [install](https://yarnpkg.com/getting-started/install) it first.

```bash
git clone git@github.com:davitJabushanuri/Chirp.git
cd Chirp
yarn install
yarn dev
```
